Tutorials

Please consider doing one of the Tutorials with screencasts.

General

  • It is often necessary to change the font size of the table to match the size of your browser window. Try using the shortcut key combination by consulting your browsers documentation (e.g. use Command,+ in Chrome on OSX to increase your font size). If the tables still look weird try using a different browser. The NRG-CING web site was developed with Chrome 14.0 on OSX and checked with: Firefox (7.0.1 on OSX 10.7.2; 6.0.2 on Ubuntu 11.04), Internet Explorer 9.0 (Windows 7), and Safari 5.1.1 (OSX and Windows)

Columns

  1. PDB entry code (click on to go to NRG-CING report).
  2. CING green, orange, red score [ 0, 1, 2] for [ good, poor, bad].
  3. Number of distance restraints from NRG.
  4. Number of chemical shifts from BMRB.
  5. Chothia definition for alpha,beta,alpha/beta,coil, and not determined [0,1,2,3,.]. This column can also be used to find entries that do not have a protein chain by selecting the period from the dropdown box below this column.
  6. Number of chains including water.
  7. Total number of residues including water.

Filtering

  • Use the Search box to search in any column of the table. Multiple words will both need to be present. E.g. enter '9pcy' to select just the row with entry 9pcy in it.
  • Use the dropdown boxes below the table to select only the few discrete values for ROG score and Chothia class.

Sorting

  • Click on the header row of the column to sort by. And again to reverse the order. Sorting on an additional column can be done by pressing the SHIF modifier key while clicking.

Pagination

  • Use the arrow buttons at the bottom right to flip through the pages.
  • Use the top left dropdown box to change the number of rows displayed.
