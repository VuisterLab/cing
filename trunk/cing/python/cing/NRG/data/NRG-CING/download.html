Plots

Many of the validation criteria in CING are also in the RDB and from that resource plots showing the distributions in histograms and trends over time are made available at: Plots.

Relational Database

In addition to the web-based interactive HTML pages we also make available CSV dumps from the relational database at: RDB. They can be imported to a slave PostgreSQL database using this SQL script. The relational database contains the validation data at the levels of entry, chain, residue and atom and tables for distance restraints and chemical shifts.
